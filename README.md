# Prueba Técnica
Puedes ver la app en la siguiente dirección:
https://app-firebase-3bc55.web.app/

## 1. Instalación

```bash
npm install
```

## 2. Levantar Proyecto modo Local
En http://localhost:3000

```bash
npm run start
```
## 3. Compilación de Estáticos

```bash
npm run public
```

## 4. Subir a Firebase Hosting

```bash
firebase deploy
```
